# Phase 2: Invoice CRUD, Cron, and Dashboard Summary

One-liner: Invoice CRUD via Convex mutations/queries with a daily cron for status transitions and a live React dashboard with filter tabs, summary cards, and mark-paid/delete actions.

## What Was Built

### convex/invoices.ts
- `create`: inserts invoice with auto-incremented `invoiceNumber` per tenant and initial status derived from `dueDate` vs today
- `list`: optional status filter using `by_owner_status` compound index
- `markPaid`: ownership check + status patch
- `remove`: ownership check + delete
- `updateStatuses`: `internalMutation` ŌĆö sweeps all non-paid invoices and transitions `upcoming ŌåÆ due` or `upcoming/due ŌåÆ overdue` based on date

### convex/crons.ts
- Daily midnight UTC job calling `internal.invoices.updateStatuses`

### convex/dashboard.ts
- `summary` query: returns `{ total, overdue, due, upcoming, unpaidAmount, paidThisMonth }` for the authenticated tenant

### Dashboard UI
- `DashboardContent.tsx`: client component with `useQuery(api.dashboard.summary)` and `useQuery(api.invoices.list)`, filter tabs (all/upcoming/due/overdue/paid), summary cards, table with Mark Paid and Delete per row
- `NewInvoiceForm.tsx`: controlled form converting dollar input to cents, date to UTC midnight timestamp, error display, redirects to dashboard on success

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] `@convex/_generated/api` import resolved to non-existent path**
- **Found during:** Task 02-03 build
- **Issue:** tsconfig `@/*` maps to `./src/*`. Adding `@convex/*` ŌåÆ `./convex/*` alias resolved at the TypeScript level, but Turbopack build failed because `convex/_generated/api.ts` did not exist on disk (only generated by `npx convex dev`).
- **Fix:** Created `convex/_generated/api.ts` stub that exports a Proxy-based `api` and `internal` object so the build resolves the module without real generated types. The stub is replaced at runtime when `npx convex dev` generates the real file.
- **Files modified:** `convex/_generated/api.ts` (created), `tsconfig.json` (added `@convex/*` path)
